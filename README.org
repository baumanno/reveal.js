** Installation
:PROPERTIES:
:ID:       8e5e8632-9a13-4280-92e3-a4141779f417
:END:

#+begin_src bash
git clone https://github.com/baumanno/reveal.js
cd reveal.js

# install dependencies
npm install

# run initial build
npm run build:theme && npm run build

# ... or run the dev server
npm run dev:all
#+end_src

* Rebasing onto ~upstream/master~
:PROPERTIES:
:ID:       cc636cba-0d46-4017-a5da-0dc58f4b7d75
:END:

This repo's ~main~ tracks ~upstream/master~. ~baumanno-theme~ contains all
commits for the custom theme. To sync these with upstream, that branch
is rebased onto ~upstream/master~.

#+begin_src bash
# initial setup only: add upstream remote
git remote add upstream https://github.com/hakimel/reveal.js.git

# update upstream
git fetch upstream

# switch to feature branch, if not already on it
git checkout baumanno-theme

# rebase current branch onto upstream/master
git rebase upstream/master

# force push the feature branch, now on top of upstream/master
git push origin baumanno-theme --force-with-lease
#+end_src
